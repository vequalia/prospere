Prospere 通過深度分析您的歷史交易路徑，為您的財富藍圖構建高精度模型。它讓財務管理不再僅僅是「記錄過去」，而是「預見未來」。

### 為您解答關鍵疑問：

*   **成長路徑**：10 年、20 年甚至 30 年後，我的資產實力會達到什麼層次？
*   **科學優化**：如何在不犧牲生活品質的前提下，找到財富增值的最優解？
*   **極端韌性**：面對 20% 的市場劇震或持續性通膨，我的財務方案是否依然穩健？
*   **選擇自由**：距離「工作成為選項，而非必需」的那一天，還有多久？

---

## 核心能力

### 1. 蒙地卡羅模擬 (Monte Carlo)
Prospere 使用 **t-分佈蒙地卡羅模擬**來考量市場波動與經濟衝擊，為您提供未來財務狀況的概率區間（P10/P50/P90）。

### 2. 行為感知優化
基於 **SLSQP（序列二次規劃）** 算法，優化引擎能精準識別預算中具備彈性的支出類別，並給出科學建議——在追求未來財富極大化的同時，將對當下生活品質的影響降至最低。

### 3. AI 財務分析師
AI 核心是進行歸類與深度分析的基礎。我們採用**模型無關 (Model-agnostic)** 的靈活架構，完整相容 OpenAI API 標準，這意味著您可以根據偏好，在 **GPT-5**、**DeepSeek** 或本地模型之間自由切換。

## 安裝指南

### 一鍵安裝 (macOS / Linux)
最簡單的安裝方式是使用我們的一鍵安裝腳本。打開你的終端機並貼上以下指令：
```bash
curl -sSL https://raw.githubusercontent.com/vequalia/prospere/main/scripts/install.sh | bash
```

### 一鍵安裝 (Windows)
打開 PowerShell 並貼上以下指令：
```powershell
irm https://raw.githubusercontent.com/vequalia/prospere/main/scripts/install.ps1 | iex
```

安裝完成後，只需打開一個**新的**終端機視窗，輸入 `prospere` 即可啟動。

### 開發者安裝 (使用 uv)
如果你想手動管理依賴或參與開發：
```bash
git clone https://github.com/vequalia/prospere.git
cd prospere
uv sync
uv run prospere
```

## 你需要準備什麼

### 交易數據

Prospere 原生支援 MoneyWiz 的 CSV 匯出檔案。如果你使用其他記帳工具（YNAB、Mint、Quicken、銀行匯出、自訂表格等），可使用我們的 **AI 驅動數據清理器 skill** 自動轉換你的數據：

```bash
# 1. 安裝 Claude Code 的 skill
cp skills/data-cleaner.zh-Hant.md ~/.claude/skills/data-cleaner.md

# 2. 用你的檔案執行
/data-cleaner path/to/your/export.csv
```

該 skill 會分析你的檔案、將欄位對應至 Prospere 格式，並產出標準輸出。你也可以自行準備好 Prospere 要求的數據格式。你需要兩個檔案：

**1. 交易記錄（`.xlsx`）**

一個 Excel 檔案，每列一筆交易：

| 欄位 | 範例 | 說明 |
|--------|---------|-------|
| `unique_id` | `a1b2c3d4...` | 每列唯一的識別字串 |
| `transaction_date` | `2025-04-03` | `YYYY-MM-DD` 格式 |
| `amount` | `48000.00` | 正數 = 收入，負數 = 支出 |
| `currency` | `HKD` | EUR、USD、CNY、ILS |
| `primary_category` | `薪資` | 主分類 |
| `secondary_category` | `本薪` | 子分類（可為空） |
| `account_name` | `HSBC 儲蓄` | 該筆交易所屬的帳戶 |

**2. 帳戶餘額（`.json`）**

一個 JSON 陣列，列出每個帳戶的當前餘額：

```json
[
    { "account_name": "HSBC 儲蓄",     "balance": 280000.00, "currency": "HKD" },
    { "account_name": "渣打基金帳戶",   "balance": 650000.00, "currency": "HKD" },
    { "account_name": "AEON 信用卡",    "balance": -8500.00, "currency": "HKD" },
    { "account_name": "富途牛牛",        "balance": 420000.00, "currency": "HKD" }
]
```

首次啟動時，從主選單選擇**匯入數據**來匯入你的 MoneyWiz CSV。如果你的數據來自其他工具，可使用 `/data-cleaner` Claude Code skill（見上方說明）轉換後，選擇**匯入預處理數據**載入清洗後的輸出。你也可以直接提供預處理好的檔案——格式參見上方範例。

### AI 配置 (必填)

AI API 金鑰是開啟核心功能的必需品，包括自動分類與 Prospere 進行互動對話。Prospere 支援任何相容於 OpenAI 的 API 端點。

當您首次啟動 `prospere` CLI 時，系統會自動引導您設定 AI 供應商（例如 OpenAI、DeepSeek 或自定義端點）並輸入您的 API 金鑰。這些設定會安全地全域儲存在 `~/.prospere_settings.json` 中，日後可隨時透過主選單的 **設定** 進行更改。

## 開始使用

一個指令。其他一切都在互動式選單中完成：

```bash
uv sync
prospere
```

### 自動更新
預設情況下，Prospere 會在啟動時檢查更新，並在檢測到 GitHub 上有新版本時，自動在您當前的 Python 環境中使用 `pip` 進行自我升級。

若要略過自動更新檢查，請設定環境變數：
```bash
export PROSPERE_NO_UPDATE=1
```

#### 場景配置

資料匯入後，從情境選單選擇**建立情境**。系統會提供兩種建立模式：

**快速建立 (Quick Setup)** — 全自動。只需指定模擬年數，其餘全部交給 AI（分類、增長建模、帳戶／類別配置、稅務規則）。最後提供完整的 Recap 總結頁面，確認後再儲存。

**引導建立 (Guided Setup)** — 逐步精靈，AI 輔助。AI 會自動將帳戶與收支類別進行分類，依據你的生涯階段建立收入／支出增長曲線，並根據你所在國家建立稅務規則。每個步驟都可以審查和修改，或使用 `--manual` 略過 AI。

```
  ✦ Prospere  ›  家明  ›  Bootstrap  ›  設定模式
  ──────────────────────────────────────────────────────────────
  ▶ 引導建立  (逐步精靈，AI 輔助)
    快速建立  (全自動 — 只需設定年數)

  ↑/↓ 導航  ·  Enter 選擇  ·  q 返回
```

### 專業級 HTML 報告

Prospere 可生成專業的 HTML 報告，專為深度財務審查而設計。你可以直接在對話 (Chat) 介面中輸入 `/export` 命令來生成並開啟這些報告。報告內容與命令行輸出完美同步，同時提供豐富的視覺化圖表與更詳盡的分析洞察。

*   **決策儀表板**：核心財務指標與成功概率一目了然，助力科學決策。
*   **視覺化圖表**：動態展示資產增長軌跡與模擬結果分佈。
*   **深度戰略分析**：針對現金流、資金效率與風險暴露進行全面剖析。
*   **多語言支援**：完美支援英文、繁體中文與簡體中文。
