# Security Policy

## Supported Versions

Please check the main branch for the latest security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in Prospere, please report it to us directly at **hello@vequalia.com**.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.
