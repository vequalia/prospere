Prospere 通过深度分析您的历史交易记录，为您的财富蓝图构建高精度模型。它让财务管理不再仅仅是「记录过去」，而是「预见未来」。

### 为您解答关键疑问：

*   **增长路径**：10 年、20 年甚至 30 年后，我的资产实力会达到什么层次？
*   **科学优化**：如何在不牺牲生活品质的前提下，找到财富增值的最优解？
*   **极端韧性**：面对 20% 的市场震荡或持续性通胀，我的财务方案是否依然稳健？
*   **选择自由**：距离「工作成为选项，而非必需」的那一天，还有多久？

---

## 核心能力

### 1. 蒙特卡洛模拟 (Monte Carlo)
Prospere 使用 **t-分布蒙特卡洛模拟**来考量市场波动与经济冲击，为您提供未来财务状况的概率区间（P10/P50/P90）。

### 2. 行为感知优化
基于 **SLSQP（序列二次规划）** 算法，优化引擎能精准识别预算中具备弹性的支出类别，并给出科学建议——在追求未来财富极大化的同时，将对当下生活品质的影响降至最低。

### 3. AI 财务分析师
AI 核心是进行归类与深度分析的基础。我们采用**模型无关 (Model-agnostic)** 的灵活架构，完整兼容 OpenAI API 标准，这意味着您可以根据偏好，在 **GPT-5**、**DeepSeek** 或本地模型之间自由切换。

## 安装指南

### 一键安装 (macOS / Linux)
最简单的安装方式是使用我们的一键安装脚本。打开你的终端并粘贴以下指令：
```bash
curl -sSL https://raw.githubusercontent.com/vequalia/prospere/main/scripts/install.sh | bash
```

### 一键安装 (Windows)
打开 PowerShell 并粘贴以下指令：
```powershell
irm https://raw.githubusercontent.com/vequalia/prospere/main/scripts/install.ps1 | iex
```

安装完成后，只需打开一个**新的**终端窗口，输入 `prospere` 即可启动。

### 开发者安装 (使用 uv)
如果你想手动管理依赖或参与开发：
```bash
git clone https://github.com/vequalia/prospere.git
cd prospere
uv sync
uv run prospere
```

## 你需要准备什么

### 交易数据

Prospere 原生支持 MoneyWiz 的 CSV 导出文件。如果你使用其他记账工具（YNAB、Mint、Quicken、银行导出、自定义表格等），可使用我们的 **AI 驱动数据清理器 skill** 自动转换你的数据：

```bash
# 1. 安装 Claude Code 的 skill
cp skills/data-cleaner.zh-Hans.md ~/.claude/skills/data-cleaner.md

# 2. 用你的文件执行
/data-cleaner path/to/your/export.csv
```

该 skill 会分析你的文件、将列映射至 Prospere 格式，并产出标准输出。你也可以自行准备好 Prospere 要求的数据格式。你需要两个文件：

**1. 交易记录（`.xlsx`）**

一个 Excel 文件，每行一笔交易：

| 字段 | 示例 | 说明 |
|--------|---------|-------|
| `unique_id` | `a1b2c3d4...` | 每行唯一的标识字符串 |
| `transaction_date` | `2025-04-03` | `YYYY-MM-DD` 格式 |
| `amount` | `28000.00` | 正数 = 收入，负数 = 支出 |
| `currency` | `CNY` | EUR、USD、CNY、ILS |
| `primary_category` | `工资` | 主分类 |
| `secondary_category` | `基本工资` | 子分类（可为空） |
| `account_name` | `招商银行储蓄卡` | 该笔交易所属的账户 |

**2. 账户余额（`.json`）**

一个 JSON 数组，列出每个账户的当前余额：

```json
[
    { "account_name": "招商银行储蓄卡",   "balance": 280000.00, "currency": "CNY" },
    { "account_name": "华泰证券A股",       "balance": 350000.00, "currency": "CNY" },
    { "account_name": "支付宝余额",        "balance": 25000.00,  "currency": "CNY" },
    { "account_name": "招商银行信用卡",    "balance": -5200.00,  "currency": "CNY" }
]
```

首次启动时，从主菜单选择**导入数据**来导入你的 MoneyWiz CSV。如果你的数据来自其他工具，可使用 `/data-cleaner` Claude Code skill（见上方说明）转换后，选择**导入预处理数据**加载清洗后的输出。你也可以直接提供预处理好的文件——格式参见上方示例。

### AI 配置 (必填)

AI API 密钥是开启核心功能的必需品，包括自动分类与 Prospere 进行交互对话。Prospere 支持任何兼容 OpenAI 的 API 端点。

当您首次启动 `prospere` CLI 时，系统会自动引导您配置 AI 供应商（例如 OpenAI、DeepSeek 或自定义端点）并输入您的 API 密钥。这些设置会安全地全局保存在 `~/.prospere_settings.json` 中，日后可随时通过主菜单的 **设置** 进行更改。

## 开始使用

一个指令。其他一切都在交互式菜单中完成：

```bash
uv sync
prospere
```

### 自动更新
默认情况下，Prospere 会在启动时检查更新，并在检测到 GitHub 上有新版本时，自动在您当前的 Python 环境中使用 `pip` 进行自我升级。

若要跳过自动更新检查，请设置环境变量：
```bash
export PROSPERE_NO_UPDATE=1
```

#### 场景配置

数据导入后，从场景菜单选择**创建场景**。系统会提供两种创建模式：

**快速创建 (Quick Setup)** — 全自动。只需指定模拟年数，其余全部交给 AI（分类、增长建模、账户／类别配置、税务规则）。最后提供完整的 Recap 总结页面，确认后再保存。

**引导创建 (Guided Setup)** — 逐步向导，AI 辅助。AI 会自动将账户与收支类别进行分类，依据你的生涯阶段建立收入／支出增长曲线，并根据你所在国家建立税务规则。每个步骤都可以审查和修改，或使用 `--manual` 跳过 AI。

```
  ✦ Prospere  ›  小明  ›  Bootstrap  ›  设置模式
  ──────────────────────────────────────────────────────────────
  ▶ 引导创建  (逐步向导，AI 辅助)
    快速创建  (全自动 — 只需设定年数)

  ↑/↓ 导航  ·  Enter 选择  ·  q 返回
```

### 专业级 HTML 报告

Prospere 可生成专业的 HTML 报告，专为深度财务审查而设计。你可以直接在对话 (Chat) 界面中输入 `/export` 命令来生成并打开这些报告。报告内容与命令行输出完美同步，同时提供丰富的交互式图表与更详尽的分析洞察。


*   **决策仪表板**：核心财务指标与成功概率一目了然，助力科学决策。
*   **交互式图表**：动态展示资产增长轨迹与模拟结果分布。
*   **深度战略分析**：针对现金流、资金效率与风险暴露进行全面剖析。
*   **多语言支持**：完美支持英文、繁体中文与简体中文。
